### Featured Projects

#### 🦅 HAWKEYE — AI-Driven Insider Fraud Detection *(2025)*

National finalist submission for the RBI NFPC Hackathon. A production-grade early-warning system that continuously monitors privileged-user behaviour across core banking, treasury, and loan-origination systems.

- **Scale:** 400M+ real banking transactions · 146 engineered features · 516K-event synthetic replay stream
- **Model:** LightGBM dual-model blend with SHAP explainability · **AUC 0.998 · F1 0.967**
- **Architecture:** 12-service Docker stack — Kafka, Redis, Neo4j, PostgreSQL, Keycloak, MLflow, Prometheus, Grafana
- **LLM Layer:** Groq `gpt-oss-120b` for analyst-grade investigation memos with deterministic Jinja fallback
- **Frontend:** Bloomberg-terminal style React 18 SPA with live WebSocket event tape, D3 heatmaps, and graph visualisations
- **Deployed live** with CI/CD via GitHub Actions, TLS via Let's Encrypt
- **Stack:** Python · FastAPI · LightGBM · SHAP · Kafka · Redis · Neo4j · PostgreSQL · React 18 · TypeScript · Docker

#### 📞 Menstrual Hygiene Survey Agent — Team Lajja *(2025)*

Multilingual outbound AI voice-calling system built for **Making the Difference NGO**. Automatically calls school-age girls who attended menstrual-hygiene awareness sessions, conducts an 8-question feedback survey in Hindi, English, or Telugu, records the call, and generates AI-driven analysis.

- **Voice Stack:** LiveKit Agents · Sarvam STT (`saaras:v3`) · Groq `gpt-oss-120b` · Sarvam TTS (`bulbul:v3-beta`)
- **Telephony:** Vobiz SIP trunk → +91 PSTN dialling · LiveKit BVCTelephony noise cancellation
- **Auto Language Detection** with debounced TTS language switching per utterance
- **Recording Pipeline:** LiveKit Room Composite Egress → MP3 → Supabase S3 with deterministic public URLs
- **Dashboard:** Next.js 16 · React 19 · real-time transcript, language badges, latency metrics over LiveKit data channel
- **Stack:** Python · LiveKit · Sarvam AI · Groq · Next.js 16 · React 19 · TypeScript · Supabase

#### 🤖 Voice-Enabled AI Business Agent Platform *(2025)*

Full-stack platform for deploying AI-powered voice call agents with real-time interaction. Six-step onboarding, multi-model AI pipeline (GPT-4o-mini, Deepgram, ElevenLabs, VAPI), and n8n automation that reduced setup time by **70%**.

**Stack:** React · Vite · Supabase · Clerk · n8n · VAPI · GPT-4o-mini

#### 🎯 Market Muse AI — Multi-Agent Marketing Workspace *(2025)*
